# Rahim Khan | Portfolio Website

A modern, responsive portfolio website for Rahim Khan — Digital Creator & Tech Enthusiast.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll animations with Intersection Observer
- Glassmorphism UI with gradient accents
- Interactive 3D parallax profile image
- Mobile-friendly navigation with hamburger menu
- Sections: Hero, About, Skills, Projects, Books, Contact
- Contact form with visual feedback
- Dark theme with animated background orbs

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript** — Vanilla JS, Intersection Observer API
- **Fonts** — Inter, Space Grotesk (Google Fonts)
- **Icons** — Font Awesome 6

## Getting Started

Simply open `index.html` in your browser, or deploy to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).
